SHEETS

Google Sheets 연동

✅ 이 시스템은 Google Apps Script를 백엔드로 사용합니다.
아래 순서대로 5분이면 설정 완료됩니다.
1
구글 시트 새로 만들기
drive.google.com → 새로 만들기 → Google 스프레드시트
이름: 데일리 보고 DB (자유롭게 설정 가능)
2
Apps Script 열기
시트 상단 메뉴 → 확장 프로그램 → Apps Script
3
코드 붙여넣기
다운로드한 앱스스크립트_코드.js 파일 내용을 전체 복사 → Apps Script 편집기에 붙여넣기 → 저장(Ctrl+S)
4
시트 초기화
Apps Script 편집기에서 함수 선택 드롭다운에서 initSheets 선택 → ▶ 실행 버튼 클릭
(처음 실행 시 권한 요청이 뜨면 허용)
5
웹 앱으로 배포
Apps Script 편집기 → 배포 → 새 배포
· 유형: 웹 앱
· 다음 사용자로 실행: 나 (본인 계정)
· 액세스 권한: 모든 사용자
→ 배포 클릭 → 웹 앱 URL 복사
6
URL 입력 후 연결 테스트
